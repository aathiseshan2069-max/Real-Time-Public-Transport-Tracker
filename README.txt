Real-Time Public Transport Tracking Starter App

Included:
- server.js
- package.json
- passenger.html
- driver_simulator.html
- db.sql

Run instructions:
1. Install Node.js & npm
2. npm install
3. Set MAPBOX_TOKEN in passenger.html and driver_simulator.html
4. node server.js
5. Open passenger.html in browser, driver_simulator.html to simulate vehicle
